What are Power BI Installation Requirements? 5

Supported Operating System

·         Windows 10, Windows 7, Windows 8, Windows 8.1, Windows Server 2008 R2, Windows Server 2012, Windows Server 2012 R2

·       Microsoft Power BI Desktop requires Internet Explorer 10 or greater


Microsoft Power BI Desktop is available for 32-bit (x86) and 64-bit (x64) platforms.


·       Users can select a language in which they want to install Power BI and following files are available for download.

Download the version of Power BI Desktop that matches the architecture (x86 or x64) of your Windows OS. Run the MSI installer and follow the setup steps. 

https://powerbi.microsoft.com/en-us/downloads/
